NETOP (network topology) is a web application that provides a visual environment for building network topologies and simulating their behavior. Inspired by the NetEmul project used in ITMO University's networking course. The frontend is built with PixiJS and Svelte, while computationally intensive tasks are handled by WebAssembly modules compiled from C++.
